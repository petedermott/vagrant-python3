Welcome to Python 3.4 on Ubuntu!

Try out Django:

(0) cd /vagrant/projects
(1) mkvirtualenv --quiet django_project
(2) pip install --quiet django==1.9
(3) django-admin.py startproject django_project
(4) cd django_project
(5) python manage.py runserver 0.0.0.0:8000

Some handy shortcuts:

Run Django's development server as above:

    $ runserver

Colored cat; syntax highlighting at the command line (try it!):

    $ ccat /vagrant/vagrant_bootstrap.sh

Just type "menu" at any time to repeat these instructions.
